# Google IT Automation With Python


This repository is created to keep track of [Google IT Automation With Python](https://www.coursera.org/professional-certificates/google-it-automation) provided by Coursera. This program is developed by Google and designed to teach how to program with Python and how to use Python to automate common system administration tasks. Also, the course will teach how to debug complex problems, and apply automation at scale by using configuration management and the Cloud.
